## 🎮 What is Slong.io?

**Slong.io** is a fast-paced, multiplayer snake arena game inspired by Slither.io — rebuilt from scratch with 3D visuals, meme energy, and chaotic mechanics that make every match unpredictable and hilarious.

Eat orbs. Grow massive. Outplay everyone. Become the **King**.

---

## ✨ Features

### 🕹️ Core Gameplay
- **Mouse-Based Controls** — Steer your snake by moving your cursor. Click to boost. Simple, satisfying, deadly
- **Real-Time Multiplayer** — Powered by Socket.io for buttery-smooth online play
- **3D Visuals** — Built with React Three Fiber and Three.js for a premium look and feel
- **AI Bots** — Never play alone. Smart bots fill the arena, hunt food, dodge danger, and keep matches intense

### 👑 Progression & Competition
- **King System** — The #1 player on the leaderboard gets crowned King with a special announcement
- **XP & Leveling** — Earn XP from collecting orbs, eliminating players, and surviving
- **Skin Unlocks** — Level up to unlock new snake skins automatically
- **Live Leaderboard** — Top 10 players displayed in real-time

### 💥 Events & Mechanics
- **MEGA FEAST** — Kill a massive snake and its body explodes into a buffet of glowing orbs
- **FOOD RUSH** — Every 45 seconds, a cluster of mega orbs spawns at a random location on the map. Race to grab them
- **Boost Trail** — Boosting sprays orbs from your head in an extreme 270° spread pattern (the "Extreme Nut Burst" 💀)
- **Wall Hug Death** — Stay too close to the boundary for 2 seconds and you're toast. No camping the edges

### 🎯 Kill Feed & Death
- **Kill Feed** — See who's eliminating who in real-time
- **Orb Drops on Death** — Every snake leaves behind a trail of collectible orbs when they die
- **Respawn System** — Hop right back in after dying. No waiting

### 🏠 Lobby & Rooms
- **Public Matchmaking** — Jump straight into a public room
- **Private Rooms** — Create or join private rooms with a custom code to play with friends
- **20 Player Cap** — Each room supports up to 20 simultaneous players for balanced, competitive gameplay

## 🚀 Run Locally

**Prerequisites:** Node.js 18+

```bash
# Clone the repo
git clone https://github.com/JustSolDev/slong-io.git
cd slong-io

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start slithering 🐍
