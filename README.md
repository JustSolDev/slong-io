# Slong.io — Neon Multiplayer Snake

A high-performance, real-time multiplayer neon snake game built with **React**, **Three.js**, and **Socket.io**. Features "Extreme Nut Burst" mechanics and AI-driven bots for a competitive ecosystem.

## 🌈 Aesthetics
- **Neon Visuals**: Custom Three.js shaders for a glowing, high-contrast aesthetic.
- **Extreme Nut Burst**: High-velocity emission system for speed boosts and tactical advantage.

## 🚀 Key Features
- **Multiplayer Hub**: Low-latency snake synchronization via Socket.io.
- **AI Bots**: Integrated bot system that simulates complex player behavior, food gathering, and tactical avoidance.
- **King Logic**: Real-time leaderboard tracking with unique "King" status and rewards.
- **Skin System**: Progression-based skin unlocking tied to player XP.

## 🛠️ Stack
- **Frontend**: React 19, Three.js, Tailwind CSS, Zustand.
- **Backend**: Express, Socket.io, Vite (Integrated mode).
- **Physics**: Shared logic for collision detection and movement types.

## 💻 Local Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start development server: `npm run dev`.
4. Open `http://localhost:3000`.
